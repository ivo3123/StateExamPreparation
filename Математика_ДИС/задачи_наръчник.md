# Основни знания за решаване на задачи по ДИС

## Основни операции с интеграли

### Интегриране по части

$$\displaystyle\int fg'\space dx = \int f\space dg = fg - \int f'g\space dx$$

### Линейност на интеграл и на диференциал

$$\displaystyle\int (a f(x) + b g(x)) \space dx = a\int f(x)\space dx + b\int g(x)\space dx$$

$$\int f \space d(ax) = \int a f \space dx = a \int f \space dx$$

### Внасяне и изнасяне от диференциала

$$f'(x)\space dx = d(f(x))$$

* При внасяне в диференциала -> интегрираме
* При изнасяне от диференциала -> диференцираме

### Адитивност на интеграл

$$\displaystyle\int_{a}^{b} f(x)\space dx = \int_{a}^{c} f(x)\space dx + \int_{c}^{b} f(x)\space dx$$

### Интеграл от четни и нечетни функции

* ако $f(x)$ е четна, тоест $f(-x) = f(x)$

$$\displaystyle\int_{-a}^{a} f(x)\space dx = 2\int_{0}^{a} f(x)\space dx$$

* ако $f(x)$ е нечетна, тоест $f(-x) = -f(x)$

$$\displaystyle\int_{-a}^{a} f(x)\space dx = 0$$

### Определен и неопределен интеграл

$$\displaystyle\int f(x)\space dx = F(x) + C \iff F'(x) = f(x)$$

$$\displaystyle\int_{a}^{b} f(x)\space dx = F(b) - F(a) \quad \text{(Формула на Нютон-Лайбниц)}$$

## Основни таблични интеграли и производни

### 1. Степенна функция (за $\alpha \ne -1$)

$$\displaystyle\int x^{\alpha}\space dx = \dfrac{x^{\alpha + 1}}{\alpha + 1} + C \quad \text{за}\space \alpha \ne -1$$

$$\left(\dfrac{x^{\alpha+1}}{\alpha+1}\right)' = x^\alpha$$

### 2. Константа

$$\displaystyle\int dx = x + C$$

$$(x)' = 1$$

### 3. Логаритмична функция

$$\displaystyle\int \dfrac{1}{x} \space dx = \mathrm{ln|x|}+ C \quad \text{за}\space x \ne 0$$

$$(\ln|x|)' = \dfrac{1}{x}$$

### 4. Показателна функция

$$\displaystyle\int \mathrm{e}^x \space dx = \mathrm{e}^x + C$$

$$(\mathrm{e}^x)' = \mathrm{e}^x$$

### 5. Синус

$$\displaystyle\int \sin(x) \space dx = -\cos(x) + C$$

$$(-\cos(x))' = \sin(x)$$

### 6. Косинус

$$\displaystyle\int \cos(x) \space dx = \sin(x) + C$$

$$(\sin(x))' = \cos(x)$$

### 7. Тангенс (през косинус квадрат)

$$\displaystyle\int \dfrac{1}{\cos^2(x)} \space dx = \tan(x) + C \quad \text{за}\space x \ne \dfrac{\pi}{2} + k\pi, k \in \mathbb{Z}$$

$$(\tan(x))' = \dfrac{1}{\cos^2(x)}$$

### 8. Котангенс (през синус квадрат)

$$\displaystyle\int \dfrac{1}{\sin^2(x)} \space dx = -\cot(x) + C \quad \text{за}\space x \ne k\pi, k \in \mathbb{Z}$$

$$(-\cot(x))' = \dfrac{1}{\sin^2(x)}$$

### 9. Аркустангенс

$$\displaystyle\int \dfrac{1}{1 + x^2} \space dx = \arctan(x) + C$$

$$(\arctan(x))' = \dfrac{1}{1 + x^2}$$

### 10. Аркуссинус

$$\displaystyle\int \dfrac{1}{\sqrt{1 - x^2}} \space dx = \arcsin(x) + C \quad \text{за}\space x \in (-1, 1)$$

$$(\arcsin(x))' = \dfrac{1}{\sqrt{1 - x^2}}$$

## Експоненти и логаритми

### Експоненти

$$a^n a^m = a^{n+m}$$

$$(a^n)^m = a^{nm}$$

$$f(x) = g(x) \iff \mathrm{e}^{f(x)} = \mathrm{e}^{g(x)}$$

### Логаритми

$$\ln(a) + \ln(b) = \ln(ab)$$

$$\ln(a) - \ln(b) = \ln\left(\dfrac{a}{b}\right)$$

$$\ln\left(x^k\right) = k\ln(x),\quad k=\text{const}$$

$$\mathrm{e}^{\ln(x)} = x$$

### Експоненти и логаритми при различни основи

$$a^x = \mathrm{e}^{x \ln(a)}$$

$$\log_a(x) = \dfrac{\ln(x)}{\ln(a)}$$

## Тригонометрия

### Основно тригонометрично тъждество

$$\sin^2(x) + \cos^2(x) = 1$$

### Понижаване на реда

$$\sin^2(x) = \dfrac{1-\cos(2x)}{2}$$

$$\cos^2(x) = \dfrac{1+\cos(2x)}{2}$$

### Двоен ъгъл

$$\sin(2x) = 2\sin(x)\cos(x)$$

$$\cos(2x) = \cos^2(x) - \sin^2(x)$$

### Тангенс/котангенс

$$\tan(x) = \dfrac{\sin(x)}{\cos(x)},\quad\cot(x) = \dfrac{\cos(x)}{\sin(x)}$$

$$\tan^2(x) = \dfrac{1}{\cos^2(x)} - 1,\quad \cot^2(x) = \dfrac{1}{\sin^2(x)} - 1$$

### Универсална субституция

$$\sin(x) = \dfrac{2 \tan\left(\dfrac{x}{2}\right)}{1 + \tan^2\left(\dfrac{x}{2}\right)}$$

$$\cos(x) = \dfrac{1 - \tan^2\left(\dfrac{x}{2}\right)}{1 + \tan^2\left(\dfrac{x}{2}\right)}$$

### Четност и нечетност

$\sin{x}$ е нечетна функция

$$\sin{(-x)} = -\sin{x}$$

$\cos{x}$ е четна функция

$$\cos{(-x)} = \cos{x}$$

### Формули за сума и разлика

$$\sin(x \pm y)=\sin(x)\cos(y)\pm\cos(x)\sin(y)$$

$$\cos(x \pm y)=\cos(x)\cos(y)\mp\sin(x)\sin(y)$$

### Изразяване чрез изместване на аргумента

$$\sin\left(\frac{\pi}{2} - x\right) = \cos(x), \quad \sin\left(\frac{\pi}{2} + x\right) = \cos(x)$$

$$\cos\left(\frac{\pi}{2} - x\right) = \sin(x), \quad \cos\left(\frac{\pi}{2} + x\right) = -\sin(x)$$

$$\dfrac{1}{(x-a)(x-b)}=\dfrac{A}{x-a}+\dfrac{B}{x-b}$$
